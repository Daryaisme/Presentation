1949 year

The first low-level programming language - Assembler was created. Now Assembler has a close relationship between language instructions and machine code. It was first widely implemented in Electronic Delay Storage Automatic Calculator(or EDSAC) which were created by Maurice Wilkes in Cambridge University in 1947. This device was used to solve differential equations and discovered a 79 digit prime number which was the largest at that time.
